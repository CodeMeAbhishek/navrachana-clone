Computer Science and Engineering (CSE) Department Courses



Semester 1

  1. Engineering Graphics

    • Course Code: CSE 101
    • Credits: 3
    • Description: Introduction to the basic concepts of engineering drawing, orthographic projections, isometric projections, and geometric constructions.
    • Prerequisites: None
    • Assessment: Assignments, Mid-term Exam, Final Exam

  2. Engineering Mathematics I

    • Course Code: CSE 102
    • Credits: 4
    • Description: Topics include linear algebra, calculus, differential equations, and mathematical methods applied to engineering problems.
    • Prerequisites: None
    • Assessment: Assignments, Mid-term Exam, Final Exam

  3. Problem Solving with Python

    • Course Code: CSE 103
    • Credits: 3
    • Description: Introduction to programming concepts using Python. Topics include variables, control structures, functions, and problem-solving techniques.
    • Prerequisites: None
    • Assessment: Assignments, Project Work, Final Exam


Semester 2

  1. Web Technology

    • Course Code: CSE 201
    • Credits: 3
    • Description: Focuses on the fundamentals of web development, including HTML, CSS, JavaScript, and the basics of backend technologies like PHP, MySQL, etc.
    • Prerequisites: Problem Solving with Python
    • Assessment: Assignments, Mid-term Exam, Final Exam

  2. Python Laboratory

    • Course Code: CSE 202
    • Credits: 1
    • Description: Hands-on lab sessions to practice Python programming, where students solve various problems and implement algorithms.
    • Prerequisites: Problem Solving with Python
    • Assessment: Lab Assignments, Lab Exam

  3. Engineering Mathematics II

    • Course Code: CSE 203
    • Credits: 4
    • Description: Advanced mathematical techniques including complex variables, vector calculus, Laplace transforms, and Fourier series.
    • Prerequisites: Engineering Mathematics I
    • Assessment: Assignments, Mid-term Exam, Final Exam


Course Summary

Engineering Graphics
Introduces basic concepts of engineering drawing.

Engineering Mathematics I

Problem Solving with Python
Introduces programming concepts using Python.

Web Technology
Teaches the fundamentals of web development with HTML, CSS, and JavaScript.

Python Laboratory
Hands-on Python programming practice and problem-solving.

Engineering Mathematics II
Includes advanced mathematical techniques such as complex variables and vector calculus.
