Winter Semester Courses

  1. Web Technology

    • Course Code: CSE 201
    • Credits: 3
    • Description: Focuses on the fundamentals of web development, including HTML, CSS, JavaScript, and the basics of backend technologies like PHP, MySQL, etc.
    • Prerequisites: Problem Solving with Python
    • Assessment: Assignments, Mid-term Exam, Final Exam

  2. Python Laboratory

    • Course Code: CSE 202
    • Credits: 1
    • Description: Hands-on lab sessions to practice Python programming, where students solve various problems and implement algorithms.
    • Prerequisites: Problem Solving with Python
    • Assessment: Lab Assignments, Lab Exam

  3. Engineering Mathematics II

    • Course Code: CSE 203
    • Credits: 4
    • Description: Advanced mathematical techniques including complex variables, vector calculus, Laplace transforms, and Fourier series.
    • Prerequisites: Engineering Mathematics I
    • Assessment: Assignments, Mid-term Exam, Final Exam
